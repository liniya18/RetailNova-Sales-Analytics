# Power BI Dashboard

This folder contains the Power BI dashboard developed for the RetailNova Sales Analytics project.

## Dashboard Features

- KPI Cards
- Monthly Sales Trend
- Sales by Region
- Sales by Category
- Top Products
- Payment Mode Analysis
- Interactive Slicers
- DAX Measures

## File

- RetailNova_Sales_Analytics.pbix

## Software Required

- Microsoft Power BI Desktop
