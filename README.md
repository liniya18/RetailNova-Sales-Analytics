# 📊 RetailNova Sales Analytics Dashboard

## 📌 Project Overview

The **RetailNova Sales Analytics Dashboard** is an end-to-end Data Analytics project developed using **Microsoft Excel, SQL, Power BI, DAX, and Power Query**. The project analyzes retail sales data to generate meaningful business insights through interactive dashboards and KPI reporting.

This project demonstrates the complete data analytics workflow, including data cleaning, exploratory analysis, SQL-based business queries, and interactive Power BI visualizations.

---

## 🎯 Business Objective

The objective of this project is to analyze retail sales data and answer key business questions such as:

- Which region generates the highest sales?
- Which product categories perform the best?
- Which payment methods are most frequently used?
- How do monthly sales trends change over time?
- What are the key performance indicators (KPIs) for business growth?
